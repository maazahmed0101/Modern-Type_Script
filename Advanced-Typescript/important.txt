🔥 npm vs npx (real difference)
🟢 1. npm = install karne ke liye

👉 npm ka kaam hota hai packages install karna

Example:

npm install typescript

👉 Ye TypeScript ko tumhare project me install karega

🟡 2. npx = run karne ke liye (without install)

👉 npx ka kaam hota hai package ko direct chalana

Example:

npx tsc --init

👉 Ye TypeScript compiler ko direct run karta hai aur tsconfig.json bana deta hai

/////___________tsconfig.js

is me ye bol raha he ke 
// File Layout
    "rootDir": "./src",
    "outDir": "./dist",

==> ye lekha he or iss ka matlab ye he ke 

Ye TypeScript ke tsconfig.json ki settings hain 👍 aur iska direct matlab folders ka structure set karna hota hai.

🔹 "rootDir": "./src"

👉 Iska matlab:
Tumhara main TypeScript code kis folder me hoga

📁 Example:
project/
 ├── src/
 │    └── index.ts

👉 Yahan:

src = tumhara TypeScript code ka folder
saari .ts files yahan hongi

🔹 "outDir": "./dist"

👉 Iska matlab:
Compiled JavaScript files kahan banengi

📁 Example:
project/
 ├── src/
 │    └── index.ts
 ├── dist/
 │    └── index.js   (auto generate hota hai)

👉 Jab tum run karte ho:

npx tsc

👉 TypeScript:

.ts ko .js me convert karta hai
aur dist folder me daal deta hai
